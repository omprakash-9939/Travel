# Design Tokens

> If a design system already defines these, link it in references.md and only record overrides here.

## Colour (by role, not raw value)
| Token | Value | Use |
|---|---|---|
| color.primary | <hex> | primary actions |
| color.surface | <hex> | backgrounds |
| color.text | <hex> | body text |
| color.danger | <hex> | errors / destructive |
| color.success | <hex> | success states |
| color.muted | <hex> | secondary text / borders |
